# Beif

Mini Project (Front-End)
ทำเฉพาะ UX/UI กับข้อมูลจำลองเท่านั้น (จินตนาการ)
เขียนด้วย .Net MAUI
คะแนนเต็ม 30 คะแนน  (ข้อละ 5 คะแนน)
เน้นการจำลองการแสดงผลตาม UX/UI ที่เหมาะสม และเปลี่ยนหน้าได้อย่างถูกต้อง
ลง GitHub URL (Public) ในการบ้านนี้
